Travel Video LLM 移动端重构草图

这不是视觉终稿，而是一张用来确认产品心智、入口优先级、页面层级和内容组织的线框板。 重点是把当前分散的功能页收拢成一个以“自动整理旅行回忆”为主叙事的移动产品。

首页主入口改为“回忆”
地图降级为辅助探索
状态反馈统一收口
详情页更像记忆册
我的页改成信任中心

这张板主要看什么

1
入口是否够直接 首次进入要不要一眼就知道主动作是“整理最近 200 张”。
2
首页是否有主心智 回忆 Hero、整理状态、回忆流三块内容有没有抢戏或冲突。
3
地图是否退到正确位置 它应该是回看空间轨迹的入口，而不是产品的主首页。
4
详情页是否更有氛围 用户点进去应该想浏览和播放，而不是进入资料管理页。
