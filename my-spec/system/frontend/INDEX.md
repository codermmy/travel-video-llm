# 前端模块索引

## 目录

- `modules/auth.md`：设备会话、token、本机身份恢复
- `modules/memories.md`：回忆主页、时间分组、快速入口
- `modules/upload.md`：导入流水线、端侧视觉队列、导入任务中心
- `modules/map.md`：地图浏览、聚类、地点补全
- `modules/story.md`：事件详情、事件编辑、照片管理、幻灯片
- `modules/profile.md`：个人资料、导入入口、数据管理

## 维护规则

1. 页面入口变化时，先更新这里的索引和代码入口。
2. 如果一个页面横跨多个服务链路，文档要明确“主链路”和“关联模块”。
3. 不再维护已经不存在的登录页、注册页、同步弹窗等旧页面说明。
