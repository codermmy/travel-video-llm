# 前端模块：个人资料与数据管理

## 1. 职责范围

- 展示本机用户信息
- 编辑昵称和头像
- 提供导入任务入口和手动导入入口
- 清理本机导入记录

## 2. 代码入口

| 类型 | 文件 |
|---|---|
| 我的页 | `mobile/app/(tabs)/profile.tsx` |
| 编辑资料 | `mobile/app/profile/edit.tsx` |
| 导入任务列表 | `mobile/app/profile/import-tasks.tsx` |
| 单任务详情 | `mobile/app/profile/import-task/[taskId].tsx` |

## 3. 我的页

### 3.1 展示内容

- 头像 / 昵称
- 本机标识 badge
- 导入记录数
- 运行中的后台任务数
- 隐私承诺状态

### 3.2 可操作项

- 进入“导入任务”
- 进入“导入照片”
- 清理导入记录

## 4. 编辑资料页

支持：

- 修改昵称
- 从相册选择头像
- 拍照上传头像

校验规则：

- 昵称只允许中文、英文、数字、下划线、连字符
- 长度 2-64
- 头像图片需小于 1MB

## 5. 当前边界

- 页面以“本机用户”视角组织，不承载多账号切换
- `username` 相关接口已存在，但当前移动端没有正式 UI 入口
- 清理的是导入记录，不会删除已经生成的事件和故事
