# 中期答辩 HTML 稿

文件：

- `index.html`：中期答辩展示稿成品，可直接浏览器打开并现场使用
- `assets/screenshots/`：可选截图目录；如果放入真实截图，会自动替换展示页中的默认成品卡片
- `assets/videos/`：可选视频资源目录

## 使用方式

1. 直接打开 `mid_pre/index.html`
2. 使用键盘方向键、`PageUp/PageDown` 或右上角按钮翻页
3. 如果要导出 PDF：
   - 浏览器选择“打印”
   - 目标选择“保存为 PDF”
   - 建议勾选“背景图形”

## 当前状态

- 现在这份稿子已经可以直接使用，不依赖额外补图或补文案
- 页面为固定 `16:9` 幻灯片结构，不是长网页
- 内容口径以学校通知、毕设要求和项目真实实现为主
