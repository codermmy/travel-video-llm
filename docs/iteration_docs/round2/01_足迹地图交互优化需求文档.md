# 足迹地图交互优化需求文档

> **文档版本**: v1.0
> **创建时间**: 2026-02-10
> **目标**: 优化足迹地图的交互体验，实现智能标记聚合和层级导航

---

## 📋 需求概述

### 当前问题
1. **标记重叠**: 同一位置或相近位置的多个事件会重叠显示，无法区分
2. **交互单一**: 无法从全国地图逐步深入到省份/城市级
3. **卡片局限**: 点击聚合点后，只能显示一个事件卡片，无法展示多个事件
4. **返回缺失**: 没有便捷的方式返回到初始地图状态

### 核心目标
- 实现基于视觉重叠的智能聚合
- 支持层级缩放导航（全国 → 省份 → 城市）
- 多事件卡片展示（堆叠或滚动列表）
- 完善的返回机制（返回上一级、返回初始状态）

---

## 🎯 核心功能需求

### 1. 智能标记聚合

**功能描述**
- 检测地图标记的视觉重叠
- 重叠的标记合并为聚合点
- 非重叠标记保持独立显示

**交互行为**
- **单击聚合点**: 显示该聚合点下的事件卡片列表
- **双击聚合点**: 地图缩放到下一级（更精细的视图）

**UI 表现**
- 基础事件点: 圆形显示事件封面图（36x36）
- 聚合点: 基础标记 + 右上角数量徽章（如 "3"）
- 选中状态: 放大 1.2 倍 + 向下箭头

---

### 2. 层级导航系统

**功能描述**
- 支持从全国地图逐级深入到具体位置
- 状态栈记录每一级的视野范围（camera position）
- 点击空白处返回上一级

**交互行为**
- **双击聚合点**: 缩放到下一级（zoom + 3）
- **单击空白处**: 返回上一级（状态栈弹出）
- **"返回初始"按钮**: 直接回到最初始的地图视野

**层级示例**
```
全国地图（所有省份聚合）
  ↓ 双击四川聚合点
四川省地图（省内城市聚合）
  ↓ 双击成都聚合点
成都市地图（具体事件）
```

**初始状态自适应**
- 如果所有事件都在四川，初始状态就是四川省地图（而非全国）
- 如果事件分布在全国，初始状态就是全国地图
- 返回按钮文字动态显示："返回全国" 或 "返回四川省"

---

### 3. 多事件卡片展示

**功能描述**
- 点击聚合点后显示该聚合点下的所有事件
- 根据事件数量选择不同的展示方式

**展示方式**

| 事件数量 | 展示方式 | 详情 |
|---------|---------|------|
| 1 个 | 单个卡片 | 显示完整事件信息 |
| 2-3 个 | 垂直堆叠 | 每个卡片完整显示，间距 8px |
| >3 个 | 滚动列表 | 初始显示 3 个，可滚动查看更多 |

**UI 设计**
- 杂志风格布局: 左侧封面图（56x56），右侧标题和元数据
- 卡片样式: 轻微阴影、16px 圆角
- 滚动提示: 顶部渐变遮罩 + "↓ 向下滚动查看更多"

**交互行为**
- **单击卡片**: 选中的卡片高亮（不进入详情）
- **再次单击卡片**: 进入事件详情页
- **点击空白处**: 清空选中

---

### 4. 返回机制

**功能描述**
- 支持两种返回方式：返回上一级、返回初始状态

**返回方式**

| 操作 | 行为 | 说明 |
|------|------|------|
| 点击空白处 | 返回上一级 | 状态栈弹出，缩放到上一级的视野 |
| "返回初始"按钮 | 返回初始状态 | 直接到状态栈底，跳过中间层级 |
| 点击卡片 | 不返回 | 仅选中/进入详情，不改变地图状态 |

**按钮设计**
- 位置: 右上角悬浮
- 样式: 胶囊形状（高度 32px，水平 padding 12px）
- 背景: rgba(255, 255, 255, 0.9) + iOS backdrop-blur
- 文字: "返回{层级名称}"（动态）

## 🏗️ 技术方案

### 方案选择：近似聚合（方案 A）

**选择理由**
- 简单、快速可用
- 不依赖额外库
- 适合当前 `react-native-amap3d` v3.2.1 的局限性

**核心逻辑**
```typescript
// 聚合阈值（约 100m）
const CLUSTER_THRESHOLD = 0.001;

// 计算两点间的 Haversine 距离（单位：公里）
function haversineDistance(lat1, lon1, lat2, lon2): number {
  const R = 6371;
  const dLat = toRadians(lat2 - lat1);
  const dLon = toRadians(lon2 - lon1);
  const a = Math.sin(dLat / 2) ** 2 +
            Math.cos(toRadians(lat1)) * Math.cos(toRadians(lat2)) *
            Math.sin(dLon / 2) ** 2;
  const c = 2 * Math.atan2(Math.sqrt(a), Math.sqrt(1 - a));
  return R * c;
}

// 聚合事件
function clusterEvents(events: EventRecord[]): EventCluster[] {
  const clusters = [];
  const processed = new Set<string>();

  for (const event of events) {
    if (processed.has(event.id)) continue;

    const nearbyEvents = [event];
    processed.add(event.id);

    for (const other of events) {
      if (processed.has(other.id)) continue;

      const dist = haversineDistance(
        event.gpsLat, event.gpsLon,
        other.gpsLat, other.gpsLon
      );

      if (dist < CLUSTER_THRESHOLD) {
        nearbyEvents.push(other);
        processed.add(other.id);
      }
    }

    // 计算聚类中心（平均经纬度）
    const centerLat = nearbyEvents.reduce((sum, e) => sum + e.gpsLat, 0) / nearbyEvents.length;
    const centerLon = nearbyEvents.reduce((sum, e) => sum + e.gpsLon, 0) / nearbyEvents.length;

    clusters.push({
      center: { latitude: centerLat, longitude: centerLon },
      events: nearbyEvents,
      count: nearbyEvents.length
    });
  }

  return clusters;
}
```

**自适应阈值策略**
```typescript
function getAdaptiveClusterThreshold(zoom: number): number {
  if (zoom >= 14) return 0.0001;  // ~10m（城市级）
  if (zoom >= 10) return 0.001;   // ~100m（区域级）
  if (zoom >= 6)  return 0.01;    // ~1km（省级）
  return 0.1;  // ~10km（国家级）
}
```

### 状态栈管理

**数据结构**
```typescript
interface CameraState {
  target: {
    latitude: number;
    longitude: number;
  };
  zoom: number;
}

interface MapViewStack {
  states: CameraState[];
  initialState: CameraState;
  currentIndex: number;
}
```

**核心操作**
```typescript
// 初始化状态栈
function initializeStack(events: EventRecord[]): MapViewStack {
  const bounds = calculateBounds(events);
  const initialState: CameraState = {
    target: {
      latitude: (bounds.minLat + bounds.maxLat) / 2,
      longitude: (bounds.minLon + bounds.maxLon) / 2
    },
    zoom: calculateZoomLevel(bounds)
  };

  return {
    states: [initialState],
    initialState,
    currentIndex: 0
  };
}

// 双击聚合点：缩放到下一级
function handleClusterDoubleClick(cluster: EventCluster, stack: MapViewStack): MapViewStack {
  const newState: CameraState = {
    target: {
      latitude: cluster.center.latitude,
      longitude: cluster.center.longitude
    },
    zoom: stack.states[stack.currentIndex].zoom + 3
  };

  return {
    ...stack,
    states: [...stack.states, newState],
    currentIndex: stack.currentIndex + 1
  };
}

// 单击空白处：返回上一级
function handleMapPress(stack: MapViewStack): MapViewStack {
  if (stack.currentIndex <= 0) return stack;

  return {
    ...stack,
    currentIndex: stack.currentIndex - 1
  };
}

// 返回初始状态
function returnToInitialState(stack: MapViewStack): MapViewStack {
  return {
    ...stack,
    states: [stack.initialState],
    currentIndex: 0
  };
}
```

const clusters = useMemo(() => {
  if (!stack) return [];
  const threshold = getAdaptiveClusterThreshold(stack.states[stack.currentIndex].zoom);
  return clusterEvents(validEvents, threshold);
}, [validEvents, stack]);

// 当前选中的聚合点

  if (!selectedClusterId) return null;
  return clusters.find(c => getClusterId(c) === selectedClusterId);

**事件处理**
```typescript
// 双击聚合点：缩放到下一级
const handleClusterDoubleClick = (cluster: EventCluster) => {
  if (!stack || !mapRef.current) return;
  const newStack = handleClusterDoubleClick(cluster, stack);
  setStack(newStack);
};

// 单击聚合点：显示卡片列表
const handleClusterPress = (cluster: EventCluster) => {
  setSelectedClusterId(getClusterId(cluster));
};

// 单击空白处：返回上一级
const handleMapPress = () => {
  if (!stack) return;

  // 如果有选中的聚合点，先清空
  if (selectedClusterId) {
    setSelectedClusterId(null);
    return;
  }

  // 否则返回上一级
  if (stack.currentIndex > 0) {
    setStack(handleMapPress(stack));
  }
};

// 返回初始状态
const handleBackToInitial = () => {
  if (!stack) return;
  setStack(returnToInitialState(stack));
  setSelectedClusterId(null);
};
```

## ⚠️ 风险与缓解

| 风险 | 缓解措施 |
|------|---------|
| 高德地图不支持坐标转换 API | 使用近似聚合（经纬度距离判断） |
| 双击检测误判 | 使用时间阈值（300ms），双击时阻止单击事件 |
| 状态栈无限增长 | 限制最大深度（如 10 层） |
| 聚合阈值不合理 | 自适应阈值（根据 zoom 级别调整） |
| 卡片列表性能问题 | 使用 `FlatList` 替代 `ScrollView` |
