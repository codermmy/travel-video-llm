# 聚类算法优化需求文档

> **文档版本**: v1.0
> **创建时间**: 2026-02-09
> **目标**: 优化照片聚类算法，提升事件识别准确度

---

## 📋 需求概述

当前项目的照片聚类算法使用简单的时空距离规则（48小时/50公里阈值），无法适应复杂的旅行场景。需要升级为更智能的混合聚类方案，结合时空密度、语义分析和时序规则，准确识别不同类型的旅行事件。

---

## 🎯 核心优化目标

### 1. 算法升级
- [ ] 从自定义 DBSCAN 升级到 HDBSCAN（层次化密度聚类）
- [ ] 实现自适应阈值机制（基于数据分布自动调整）
- [ ] 支持层级聚类结果（可选不同粒度）

### 2. 语义增强
- [ ] 集成 AI 照片描述用于语义相似度分析
- [ ] 使用 CLIP embeddings 计算照片语义距离
- [ ] 实现语义修正机制（合并语义相似但时空相邻的簇）

### 3. 规则优化
- [ ] 基于实际旅行模式的时序规则
- [ ] 支持场景识别（城市内旅行、跨城市旅行、公路旅行）
- [ ] 处理边界情况（夜间拍照、过境点、短时间间隔）

### 4. 性能与扩展性
- [ ] 取消单事件50张照片的上限
- [ ] 支持大规模照片集（1000+张照片）
- [ ] 优化聚类算法性能

## 🏗️ 技术方案设计

### 架构概览

```
输入：用户上传照片集（GPS坐标、时间戳、AI描述）
         ↓
┌─────────────────────────────────────────────────────┐
│  Layer 1: 时空密度聚类（初步分组）          │
│  - HDBSCAN 算法                             │
│  - 自适应 eps（基于时间/地理密度）            │
│  - 层级结构输出                              │
└─────────────────────────────────────────────────────┘
         ↓
┌─────────────────────────────────────────────────────┐
│  Layer 2: 语义相似度修正（同事件合并）        │
│  - CLIP embeddings 计算                       │
│  - 聚类间语义相似度分析                   │
│  - 合并语义相似的相邻簇                      │
└─────────────────────────────────────────────────────┘
         ↓
┌─────────────────────────────────────────────────────┐
│  Layer 3: 时序规则增强（边界优化）           │
│  - 时间间隔规则                              │
│  - 地理跳跃检测                              │
│  - 场景识别规则                              │
└─────────────────────────────────────────────────────┘
         ↓
输出：事件列表（每个事件关联一组照片）
```

### Layer 1: 时空密度聚类（HDBSCAN）

**为什么选择 HDBSCAN？**

| 特性 | 当前 DBSCAN | HDBSCAN（推荐） |
|------|-------------|------------------|
| 阈值预设 | 固定 eps=1.0 | 无需预设，自适应密度 |
| 噪声处理 | 敏感度固定 | 基于密度自动识别 |
| 层级结构 | 不支持 | ✅ 支持多粒度输出 |
| 形状适应性 | 规则形状 | 任意形状簇 |
| 参数调优 | 手动调整 eps | 半自动（min_cluster_size） |

**实现要点：**

```python
from hdbscan import HDBSCAN
from sklearn.metrics.pairwise import haversine_distances

class SpacetimeHDBSCAN:
    def __init__(self, min_cluster_size: int = 5):
        """
        最小簇大小：默认5张照片（保持当前配置）
        可根据照片总量自适应调整
        """
        self.clusterer = HDBSCAN(
            min_cluster_size=min_cluster_size,
            metric='precomputed',  # 使用自定义距离矩阵
            cluster_selection_method='eom',  # 基于稳定性选择簇
            allow_single_cluster=False  # 强制分割
        )

    def _build_distance_matrix(self, photos: list[PhotoData]) -> np.ndarray:
        """
        构建时空距离矩阵

        距离公式（加权欧氏距离）：
        - 时间维度：归一化到 [0, 1]
        - 空间维度：Haversine 距离（公里），归一化到 [0, 1]
        - 综合：sqrt(time_norm² + spatial_norm²)

        权重可根据场景调整（城市内旅行：时间权重高；跨城市：空间权重高）
        """
        n = len(photos)
        matrix = np.zeros((n, n))

        time_threshold = self._adaptive_time_threshold(photos)
        distance_threshold = self._adaptive_distance_threshold(photos)

        for i in range(n):
            for j in range(i + 1, n):
                time_diff = abs(photos[i].shoot_time - photos[j].shoot_time)
                time_norm = min(time_diff.total_seconds() / (time_threshold.total_seconds()), 1.0)

                spatial_dist = 0.0
                if (photos[i].gps_lat and photos[i].gps_lon and
                    photos[j].gps_lat and photos[j].gps_lon):
                    spatial_dist = haversine(
                        (photos[i].gps_lat, photos[i].gps_lon),
                        (photos[j].gps_lat, photos[j].gps_lon),
                        unit='km'
                    )
                    spatial_norm = min(spatial_dist / distance_threshold, 1.0)

                combined = math.sqrt(time_norm * time_norm + spatial_norm * spatial_norm)
                matrix[i, j] = matrix[j, i] = combined

        return matrix

    def _adaptive_time_threshold(self, photos: list[PhotoData]) -> timedelta:
        """
        基于照片时间间隔自适应时间阈值

        策略：使用 75 分位数作为基准
        - 紧密拍照场景（如景点游览）：时间间隔小，阈值小
        - 稀疏拍照场景（如公路旅行）：时间间隔大，阈值大
        """
        if len(photos) < 10:
            return timedelta(hours=48)  # 保守默认值

        time_gaps = []
        sorted_photos = sorted(photos, key=lambda p: p.shoot_time)

        for i in range(1, len(sorted_photos)):
            gap = sorted_photos[i].shoot_time - sorted_photos[i - 1].shoot_time
            time_gaps.append(gap.total_seconds())

        threshold_seconds = np.percentile(time_gaps, 75)
        return timedelta(seconds=threshold_seconds)

    def _adaptive_distance_threshold(self, photos: list[PhotoData]) -> float:
        """
        基于照片地理密度自适应距离阈值

        策略：使用 80 分位数作为基准
        - 城市内密集拍照：距离小，阈值小
        - 乡村/公路拍照：距离大，阈值大
        """
        photos_with_gps = [p for p in photos if p.gps_lat and p.gps_lon]

        if len(photos_with_gps) < 10:
            return 50.0  # 保守默认值（公里）

        distances = []
        for i in range(len(photos_with_gps)):
            for j in range(i + 1, min(i + 6, len(photos_with_gps))):
                dist = haversine(
                    (photos_with_gps[i].gps_lat, photos_with_gps[i].gps_lon),
                    (photos_with_gps[j].gps_lat, photos_with_gps[j].gps_lon),
                    unit='km'
                )
                distances.append(dist)

        threshold_km = np.percentile(distances, 80)
        return threshold_km
```

### Layer 2: 语义相似度修正

**设计目标：**
- 区分同一地点的不同主题（如：风景 vs 美食）
- 合并主题相同但时空分散的照片（如：同一景点的不同角落）
- 过滤与旅行主题无关的照片（如：证件照、物品照）

**实现方案：**

```python
from sentence_transformers import SentenceTransformer
import torch

class SemanticClustering:
    def __init__(self):
        # 使用 CLIP 模型（中文支持好的版本）
        self.model = SentenceTransformer('clip-ViT-B-32-multilingual-v1')

    def get_representative_photos(self, cluster: list[PhotoData]) -> list[PhotoData]:
        """
        从簇中选择代表性照片

        策略：
        - 10+张照片：选 5 张（前、中、后 + 随机2张）
        - 5-10张：选 3 张
        - <5张：选全部
        """
        sorted_photos = sorted(cluster, key=lambda p: p.shoot_time)

        if len(cluster) <= 5:
            return sorted_photos

        indices = [0, len(cluster) // 2, len(cluster) - 1]
        # 随机补充 2 张中间位置的照片
        middle_start, middle_end = len(cluster) // 4, 3 * len(cluster) // 4
        random_indices = random.sample(range(middle_start, middle_end), 2)

        selected_indices = sorted(set(indices + random_indices))
        return [sorted_photos[i] for i in selected_indices]

    def compute_cluster_embeddings(self, clusters: list[list[PhotoData]]) -> dict[int, np.ndarray]:
        """
        为每个簇计算语义 embedding（基于代表性照片）
        """
        cluster_embeddings = {}

        for cluster_id, photos in enumerate(clusters):
            representatives = self.get_representative_photos(photos)

            # 为每张代表照片计算 embedding
            embeddings = []
            for photo in representatives:
                # 获取 AI 描述（临时生成，不持久化）
                description = self._get_ai_description(photo)

                # 拼接描述 + 时间+位置上下文
                context = f"{description} 于 {photo.shoot_time} 在 {photo.gps_lat},{photo.gps_lon}"

                with torch.no_grad():
                    embedding = self.model.encode(context, convert_to_tensor=True)
                    embeddings.append(embedding.cpu().numpy())

            # 聚合：平均值
            cluster_embeddings[cluster_id] = np.mean(embeddings, axis=0)

        return cluster_embeddings

    def find_semantic_similar_pairs(self, cluster_embeddings: dict[int, np.ndarray], threshold: float = 0.85) -> list[tuple[int, int]]:
        """
        找到语义相似的簇对

        使用余弦相似度
        """
        similar_pairs = []

        for i, emb_i in cluster_embeddings.items():
            for j, emb_j in cluster_embeddings.items():
                if i >= j:
                    continue

                # 余弦相似度
                similarity = np.dot(emb_i, emb_j) / (np.linalg.norm(emb_i) * np.linalg.norm(emb_j))

                if similarity >= threshold:
                    similar_pairs.append((i, j, similarity))

        # 按相似度降序排序
        similar_pairs.sort(key=lambda x: x[2], reverse=True)
        return similar_pairs

    def merge_semantic_clusters(
        self,
        clusters: list[list[PhotoData]],
        similar_pairs: list[tuple[int, int, float]]
    ) -> list[list[PhotoData]]:
        """
        合并语义相似的相邻簇

        约束：
        - 只合并时空相邻的簇（不合并距离远的）
        - 合并后保持时间顺序
        """
        if not similar_pairs:
            return clusters

        # 并查集追踪合并
        parent = {i: i for i in range(len(clusters))}

        def find(x):
            if parent[x] != x:
                parent[x] = find(parent[x])
            return parent[x]

        def union(x, y):
            px, py = find(x), find(y)
            if px != py:
                parent[px] = py

        # 合并语义相似的簇
        for i, j, similarity in similar_pairs:
            if self._are_spatially_adjacent(clusters[i], clusters[j]):
                union(i, j)

        # 按根节点分组
        merged = {}
        for i in range(len(clusters)):
            root = find(i)
            if root not in merged:
                merged[root] = []
            merged[root].extend(clusters[i])

        # 重新排序
        result = []
        for cluster in merged.values():
            result.append(sorted(cluster, key=lambda p: p.shoot_time))

        return result

    def _are_spatially_adjacent(self, cluster_a: list[PhotoData], cluster_b: list[PhotoData]) -> bool:
        """
        判断两个簇是否空间相邻

        条件：最近的两张照片距离 < 自适应阈值
        """
        if not cluster_a or not cluster_b:
            return False

        a_gps = [(p.gps_lat, p.gps_lon) for p in cluster_a if p.gps_lat and p.gps_lon]
        b_gps = [(p.gps_lat, p.gps_lon) for p in cluster_b if p.gps_lat and p.gps_lon]

        if not a_gps or not b_gps:
            return False

        min_dist = float('inf')
        for lat_a, lon_a in a_gps:
            for lat_b, lon_b in b_gps:
                dist = haversine((lat_a, lon_a), (lat_b, lon_b), unit='km')
                min_dist = min(min_dist, dist)

        # 自适应阈值（取两簇平均密度的80分位）
        all_photos = cluster_a + cluster_b
        threshold = self._adaptive_distance_threshold(all_photos) * 2  # 2倍缓冲

        return min_dist <= threshold

    def _get_ai_description(self, photo: PhotoData) -> str:
        """
        获取照片的 AI 描述（临时生成）

        不持久化到数据库，仅用于聚类时的语义分析
        """
        # 调用现有的 ai_service
        from app.services.ai_service import ai_service

        # 使用已有的照片描述逻辑
        # 从缓存或实时生成
        return ai_service.get_photo_description(photo.id)
```

### Layer 3: 时序规则增强

**规则类型：**

```python
class TemporalRules:
    """
    基于旅行模式的时序规则集
    """

    @staticmethod
    def merge_short_intervals(events: list[list[PhotoData]], min_interval: timedelta = timedelta(minutes=30)) -> list[list[PhotoData]]:
        """
        规则1：短时间间隔合并

        如果两个事件时间间隔 < 30分钟，强制合并
        （如：同一个景区的不同景点游览）
        """
        if len(events) < 2:
            return events

        events = sorted(events, key=lambda e: e[0].shoot_time)
        merged = []

        for event in events:
            if not merged:
                merged.append(event)
                continue

            last_event = merged[-1]
            last_photo = last_event[-1]
            first_photo = event[0]

            time_gap = first_photo.shoot_time - last_photo.shoot_time

            if time_gap < min_interval:
                # 合并
                merged[-1] = last_event + event
            else:
                merged.append(event)

        return merged

    @staticmethod
    def split_city_transitions(events: list[list[PhotoData]], jump_threshold: float = 100) -> list[list[PhotoData]]:
        """
        规则2：城市转换分割

        如果GPS发生大跳跃（>100km），强制分割事件
        （如：成都 → 重庆的高铁旅行）
        """
        if len(events) < 2:
            return events

        result = []
        current_event = events[0]

        for event in events[1:]:
            last_photo = current_event[-1]
            first_photo = event[0]

            if (last_photo.gps_lat and last_photo.gps_lon and
                first_photo.gps_lat and first_photo.gps_lon):

                distance = haversine(
                    (last_photo.gps_lat, last_photo.gps_lon),
                    (first_photo.gps_lat, first_photo.gps_lon),
                    unit='km'
                )

                if distance > jump_threshold:
                    # 分割
                    result.append(current_event)
                    current_event = event
                else:
                    # 合并
                    current_event = current_event + event
            else:
                # 无GPS，保守合并
                current_event = current_event + event

        result.append(current_event)
        return result

    @staticmethod
    def identify_travel_segments(events: list[list[PhotoData]]) -> dict[str, any]:
        """
        规则3：旅行阶段识别

        识别以下阶段类型：
        - city_tour: 城市内游览（GPS密集变化小）
        - transit: 交通途中（GPS沿路径移动，照片稀疏）
        - stopover: 中途停留（短暂拍照，<10张）
        - scenic: 景点游览（拍照密集，>20张）
        """
        segments = []

        for event in events:
            if len(event) < 5:
                segments.append({'event': event, 'type': 'stopover'})
                continue

            # 计算GPS覆盖范围
            gps_points = [(p.gps_lat, p.gps_lon) for p in event if p.gps_lat and p.gps_lon]
            if not gps_points:
                segments.append({'event': event, 'type': 'unknown'})
                continue

            lats = [lat for lat, _ in gps_points]
            lons = [lon for _, lon in gps_points]

            lat_range = max(lats) - min(lats)
            lon_range = max(lons) - min(lons)

            time_span = event[-1].shoot_time - event[0].shoot_time
            photo_density = len(event) / (time_span.total_seconds() / 3600)  # 张/小时

            if photo_density > 5:  # >5张/小时
                if lat_range < 0.01 and lon_range < 0.01:  # <1km范围
                    segments.append({'event': event, 'type': 'scenic'})
                else:
                    segments.append({'event': event, 'type': 'city_tour'})
            else:
                if lat_range > 0.1 or lon_range > 0.1:  # >10km范围
                    segments.append({'event': event, 'type': 'transit'})
                else:
                    segments.append({'event': event, 'type': 'stopover'})

        return segments

    @staticmethod
    def filter_night_photos(events: list[list[PhotoData]]) -> list[list[PhotoData]]:
        """
        规则4：夜间照片隔离

        单张照片在深夜（22:00-06:00）且无邻近照片 → 独立事件或噪声
        """
        night_events = []

        for event in events:
            if len(event) == 1:
                photo = event[0]
                hour = photo.shoot_time.hour

                if 22 <= hour or hour <= 6:
                    # 夜间单张：标记为独立事件
                    night_events.append(event)
                else:
                    # 白天单张：可能是噪声，但保留
                    night_events.append(event)
            else:
                night_events.append(event)

        return night_events
```

  - 19:00-20:00：火锅店（7张，GPS: 30.68°N, 104.07°E，距离1.2km）

**预期结果**
- 4-5个事件：
  - 事件1：熊猫基地（景点）
  - 事件2：春熙路（美食/商业）

- ✅ 总照片数55张（无50张限制）

**失败场景（旧算法）**
- ❌ 由于时间跨度11小时<48h，可能合并成一个大事件
- ❌ 无法识别美食vs景点的语义差异

### A. 距离计算公式

```python
from math import radians, cos, sin, asin, sqrt

def haversine(lat1, lon1, lat2, lon2):
    """
    Haversine 公式计算两点间距离（单位：公里）
    """
    R = 6371  # 地球半径（公里）

    lat1, lon1, lat2, lon2 = map(radians, [lat1, lon1, lat2, lon2])

    dlat = lat2 - lat1
    dlon = lon2 - lon1

    a = sin(dlat/2)**2 + cos(lat1) * cos(lat2) * sin(dlon/2)**2
    c = 2 * asin(sqrt(a))

    return R * c
```

### B. 余弦相似度计算

```python
def cosine_similarity(embedding_a, embedding_b):
    """
    计算两个embedding的余弦相似度
    返回：[0, 1] 区间，1为完全相同
    """
    dot_product = np.dot(embedding_a, embedding_b)
    norm_a = np.linalg.norm(embedding_a)
    norm_b = np.linalg.norm(embedding_b)

    return dot_product / (norm_a * norm_b)
```
