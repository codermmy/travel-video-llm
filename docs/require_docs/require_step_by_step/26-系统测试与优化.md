## 3. 优化建议

### 3.1 性能优化

| 优化项 | 方案 |
|--------|------|
| 图片加载 | 懒加载 + 本地缓存 |
| 列表渲染 | FlatList 虚拟列表 |

| 缓存策略 | Redis 缓存热点数据 |
