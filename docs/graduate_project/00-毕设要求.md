主要研究内容 
每个人的手机相册中都存有海量照片，但大多杂乱无章，被永久封存。本项目旨在研发一款手机APP, 自动发现并聚合相册中的零散旅行片段，将其串联成专属的旅程影片和旅途回忆。主要包括：  
时空聚类：基于照片的EXIF信息（拍摄时间、GPS坐标），自动将同一时间、地点密集拍摄的照片聚类为一个“事件片段” 。  
故事线编排：按照时间顺序（旅程的天然叙事线）自动组织这些片段。系统可为每个片段生成一个描述性标题（利用图像描述生成模型，如BLIP）。  
音乐生成：分析每个“事件片段”的整体视觉情感基调，将感知到的情感标签（如“欢快”、“宁静”、“史诗感”）作为输入，驱动一个AI音乐生成模型，创作出与照片情感相匹配的、独一无二的音乐配乐。  
旅行轨迹可视化：提取所有照片的GPS轨迹点，在地图（如百度地图API）上绘制出完整的旅行路径。在路径上标记出各个“事件片段”的发生地点，并可以点击查看该片段的封面或简短描述，形成一张动态的“旅行足迹图”。

目标和要求
目标：该系统能够全自动地对用户旅行相册中的零散照片和视频进行多模态分析、情感感知与叙事化重构，最终输出一部配有个性化原创配乐和旅行轨迹图的专属旅程影片，旨在提升用户旅行记忆的情感价值与艺术表现力。
要求：  
系统需支持常见格式（JPEG, PNG）的媒体文件上传。  
从图像文件中准确提取EXIF元数据，特别是拍摄时间和GPS坐标。  
根据时间和地理位置坐标对媒体进行分组。  
调研现有的基于情感生成音乐的模型并进行复现

特色
