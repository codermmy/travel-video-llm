# 面向旅途的智能 VLOG 创作应用参考文献调研（已核验）

## 1. 说明

这份文档面向论文 [`面向旅途的智能VLOG创作应用设计与实现`](./面向旅途的智能VLOG创作应用设计与实现_论文初稿.docx) 当前稿件整理，目标是给出一组**已经正式发表**、且与本项目实际实现强相关的文献，供论文正文和文末参考文献使用。

本次筛选以你的系统真实能力为准，重点覆盖以下五条主线：

1. 旅行照片/手机相册的自动组织与事件聚合
2. 时空聚类、照片事件划分与相册结构化
3. 端侧视觉、移动端轻量视觉模型、OCR 相关能力
4. 多图像叙事生成、相册摘要与故事生成
5. 幻灯片/视频摘要/关键帧与关键图片选择

## 2. 核验标准

只保留以下文献：

- 已发表在**正式期刊**或**正式会议论文集**中的论文
- 能查到**期刊/会议名称、年份、DOI 或正式出版页面**
- 与你的系统功能链路存在明确对应关系

明确**不保留**：

- 只有 arXiv / CoRR / 技术博客、没有正式发表记录的预印本
- 与项目关系过弱、只能“沾边”的泛 AI 文献

## 7. 使用建议

- 你的论文不是“算法论文”，而是“设计与实现”。因此引用时不要假装自己复现了这些论文方法。
