# 旅行相册自动聚合与回忆生成系统 - 中期答辩 PPT (HTML 版)

这是一个使用 HTML/CSS/JS 开发的答辩演示文稿，专为中期答辩设计。

## 使用说明

1. **直接打开**：使用 Chrome 或 Edge 浏览器打开目录下的 `index.html`。
2. **演示操作**：
   - **下一页**：按下 `方向键右 (→)`、`方向键下 (↓)` 或 `空格键 (Space)`。
   - **上一页**：按下 `方向键左 (←)` 或 `方向键上 (↑)`。
3. **视觉比例**：
   - 页面固定为 **16:9** (1280x720)。
   - 建议在浏览器中按下 `F11` 进入全屏模式以获得最佳演示效果。

## 目录结构

- `index.html`: 完整的 PPT 内容与样式。
- `README.md`: 本说明文件。

## 核心内容回顾

1. 封面 (马茂元, 武蕾, 2026.04)
2. 课题背景与目标
3. 系统总体方案
4. 核心流程 (导入-同步-聚类-生成-导出)
5. 已完成工作 (主链路已通)
6. 界面结构
7. 结果展示 (占位，可替换)
8. 阶段结论
9. 当前主要问题 (聚类精细度、身份体系、性能)
10. 下一阶段计划

## 视觉风格说明

- **克制美感**：采用森林绿与天空蓝的旅行色调，风格清新且成熟。
- **去除 AI 味**：避免了常见的圆角堆砌和廉价科技感设计，更接近 editorial/clean presentation 风格。
- **成品感**：即使不替换截图，页面也具备极强的可讲性和完整性。
