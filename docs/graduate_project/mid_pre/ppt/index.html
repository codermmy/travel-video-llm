01 研究背景与核心价值

问题定义
  • 数据冗余：个人移动终端积累了海量非结构化影像数据。
  • 整理成本：传统手动筛选与编辑流程繁琐，导致旅行情感价值“封存”。
核心价值
  • 时空聚合：将离散的照片自动化转化为有意义的“事件流”。
  • 叙事重构：利用多模态 AI 技术提取视觉语义并生成叙事文本。
  • 动态呈现：结合地理轨迹与情感匹配音乐，实现沉浸式回忆。

02 系统总体方案设计

系统架构设计理念
  • 单设备优先 (Local-First)：基于移动端原生 AssetID 管理，减少流量开销并保护用户隐私。
  • 混合处理流水线：端侧进行视觉特征分析，云端负责复杂的 LLM 叙事生成。
  • 异步一致性：采用 Celery + Redis 构建任务中台，确保海量媒体处理的稳定性。
Tech Stack: Expo 54 | React Native 0.81 | FastAPI | DeepSeek

07 存在的问题与技术瓶颈

聚类算法长尾效应

在复杂的长途跨省旅行场景中，现有的聚类参数可能导致事件“划分过细”或“合并错误”。

导出性能开销

在低性能移动设备上，原生导出渲染存在内存抖动，需要进一步优化多媒体合成流水线。

同步鲁棒性

当本地照片库发生外部物理删除时，系统内的 Metadata 缓存与本地 AssetID 的映射同步机制尚不够智能，需引入 Watcher 监听。
