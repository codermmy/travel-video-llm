# 管理员重聚类接口与脚本

> 适用目录：`/Users/maoyuan/code/travel-video-llm/backend`

## 1) 目标

用于在算法升级后，对历史数据进行 **批量重算聚类**：
- 重置用户历史事件与照片归属
- 重新执行聚类
- （可选）重新做地理逆编码

---

## 2) 接口方式（管理员）

### 2.1 配置管理员密钥
在 `backend/.env` 增加：

```env
ADMIN_API_KEY=your-admin-secret
```

重启后端服务后生效。

### 2.2 接口地址
- `POST /api/v1/admin/recluster`
- Header: `X-Admin-Key: your-admin-secret`

### 2.3 请求体

#### A. 重算单个用户
```json
{
  "userId": "用户ID",
  "runGeocoding": false
}
```

#### B. 重算全部用户（可限量）
```json
{
  "allUsers": true,
  "limitUsers": 100,
  "runGeocoding": false
}
```

### 2.4 curl 示例

```bash
curl -X POST http://127.0.0.1:8000/api/v1/admin/recluster \
  -H 'Content-Type: application/json' \
  -H 'X-Admin-Key: your-admin-secret' \
  -d '{"userId":"<USER_ID>","runGeocoding":false}'
```

```bash
curl -X POST http://127.0.0.1:8000/api/v1/admin/recluster \
  -H 'Content-Type: application/json' \
  -H 'X-Admin-Key: your-admin-secret' \
  -d '{"allUsers":true,"limitUsers":200,"runGeocoding":false}'
```

## 3) 脚本方式（推荐日常运维）

新增脚本：
- `backend/scripts/admin_recluster.py`

### 3.1 先看目标用户（dry-run）

```bash
cd /Users/maoyuan/code/travel-video-llm/backend
source .venv/bin/activate
python scripts/admin_recluster.py --all-users --limit-users 20 --dry-run
```

### 3.2 重算单个用户

```bash
cd /Users/maoyuan/code/travel-video-llm/backend
source .venv/bin/activate
python scripts/admin_recluster.py --user-id <USER_ID>
```

### 3.3 批量重算全部用户

```bash
cd /Users/maoyuan/code/travel-video-llm/backend
source .venv/bin/activate
python scripts/admin_recluster.py --all-users
```

### 3.4 批量重算 + 限量 + 地理编码

```bash
cd /Users/maoyuan/code/travel-video-llm/backend
source .venv/bin/activate
python scripts/admin_recluster.py --all-users --limit-users 100 --run-geocoding
```

### 3.5 JSON 输出（方便存档）

```bash
cd /Users/maoyuan/code/travel-video-llm/backend
source .venv/bin/activate
python scripts/admin_recluster.py --all-users --limit-users 50 --json
```

---

## 4) 安全与注意事项

1. 该操作会 **重置并重建事件聚类结果**（面向用户全量重算）。
2. 建议先在测试环境或先 `--limit-users` 小批量验证。
3. 对线上库执行前建议先备份。
4. 管理员接口必须设置 `ADMIN_API_KEY`，否则接口会返回 `admin_api_key_not_configured`。
